# Flight Price Prediction App ✈️
ML-Powered Flight Price Prediction App An end-to-end machine learning project that includes data preprocessing, model training, and deployment using Streamlit. The application loads a trained .pkl model to predict flight prices based on multiple travel features. Designed for interactive and real-time inference.

## Tech Stack
- Python
- Streamlit
- scikit-learn
- Pandas
- NumPy

## Features
- User-friendly web interface
- Real-time price prediction
- Trained ML model using historical data

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
